# Budget-Mate
------V1.0------

A fullstack web application that allows users to track their income and expenses, visualize data in charts, and manage transactions. Each user has a private account with their own transactions.

Built using HTML, JavaScript, Node.js, and MySQL.

Features :

User Authentication: Sign up and log in with a unique username.

Multi-User Support: Each userâ€™s transactions are isolated using user_id.

CRUD Operations on Transactions: Create, read, delete.

Dynamic Charts: Doughnut chart for visualizing income vs. expenses.

Auto-Increment IDs: Database handles unique transaction IDs tied to users ids.

Responsive and Interactive: Frontend updates dynamically on user actions.


In order to run you need to :
Have Node Modules installed in to the Directory
Create a Database (local or not)
Run node server. js
Run index.html

GOOD TO GO!!!
